Description détaillée du Projet

Le but du jeu est d'allumer toutes les lumières en un minimum de coups. Comment allumer les lumières ? En cliquant sur les cases. Mais attention, chaque case allumée éteint les cases adjacentes. Il faut donc trouver le bon ordre pour allumer toutes les lumières.

Partie technique

Le jeu est réalisé en C# avec le framework Unity. Le jeu est disponible sur Android et sur Windows. Il y a également un système de sauvegarde permettant de sauvegarder la progression du joueur et la mettre sur un leaderboard à but challengeant.
