Projet 2: Nom caché, application web /!\ Le projet est en cours de réalisation

L'entreprise souhaite rester anonyme. Cette dernière travaille dans la pédagogie de la finance. Les utilisateurs peuvent créer leur compte et suivre des cours upload par l'entreprise. Ils peuvent également poser des questions aux formateurs et aux autres utilisateurs. Avec leur abonnement ils possèdent également un compte propfirm et des heures de rendez-vous de mentoring.

Page d'accueil

La page d'accueil permet de voir l'état du compte de l'utilisateur, les actualités postées, les derniers messages postés dans le chat. Il y a également un modal pour voir l'état du portefeuille de l'utilisateur.

Prise de rendez-vous

Les utilisateurs peuvent prendre des rendez-vous sur les créneaux disponibles. Ils ont le choix entre un appel basic ou un appel de mentoring.

Cours

Les utilisateurs peuvent suivre des cours en ligne sous format PDF ou vidéo. Ils peuvent le télécharger ou le regarder directement sur le site.

Partie technique

Le frontend est réalisé en HTML, CSS et JS. Le backend est réalisé en Python avec le framework Flask. Il y a une base de données SQL pour stocker les informations des utilisateurs. Le site est sécurisé avec un système de token. Les mots de passes sont hashés et les données sensibles sont cryptées. Le site est responsive et est compatible avec les navigateurs les plus utilisés. Le chat est réalisé avec Socket.io.
