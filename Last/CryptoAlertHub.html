Projet 1: CryptoAlertHub

CryptoAlertHub allows you to send notification emails based on your alerts configured on your indicators.

Description détaillée du Projet

Le projet consiste à informer les utilisateurs sur les variations du marché crypto leur permettant si il y a un moment propice pour investir ou vendre sans même regarder le marché en permanence. Ils peuvent également se servir de ce site pour juste profiter des informations globales des cryptomonnaies comme leur prix, leur capitalisation, leur volume d'échange, etc...

Description détaillée du Projet

Les utilisateurs peuvent s'inscrire et se connecter pour pouvoir créer des alertes sur les cryptomonnaies qu'ils souhaitent.

Description détaillée du Projet

Les utilisateurs peuvent également consulter les alertes qu'ils ont créé et les supprimer si ils le souhaitent.

Description détaillée du Projet

Il y a également un chat pour chaque crypto permettant aux utilisateurs de discuter entre eux.

Partie technique

Le frontend est réalisé en HTML, CSS et JS. Le backend est réalisé en Python avec le framework Flask. La base de données est hashé avec le module werkzeug.
