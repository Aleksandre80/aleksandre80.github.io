Projet 3: Hypercraft

Ce projet consiste a faire la promo d'un serveur Minecraft.

Description détaillée du Projet

Le projet consiste à présenter le serveur et ses fonctionnalités aux joueurs.

Partie technique

Le frontend est réalisé en HTML, CSS et JS. Le backend est réalisé en Python avec le framework Flask. Le site communique directement avec le serveur pour recevoir des informations par requêtes POST et GET. Les informations communiquées permettent de mettre à jour le site en temps réel comme pour la liste des staffs, le nombre de votes des joueurs ainsi que le délai qu'ils leurs restent. Il y a également un panel administrateur complet permettant au client de pouvoir gérer le site lui même comme les actualités, les règles, les FAQ, les recrutements du staff etc... Le site sera bientôt en ligne
